Maya たまに使う機能

2021-01-05
  • Maya

以下列挙 & 随時追記

テクスチャ表示に対応させるUVsetを指定する

Windows > Relationship Editors > UV Linking > Texture-Centric

UV-CenticはUVsetを起点として指定する。どっちでもいいと思う。

UVset2番目にディフューズを表示させたいものを配置した場合、これを行う必要がある場合あり。基本的に1番めにディフューズ向けのUV配置してるだろうからめったに無いけど…。

骨の方向を変える

SkeletonメニューにRerootというものがあるので、それで親子関係を反転できる。Rootにしたい方の骨を選択して実行。
